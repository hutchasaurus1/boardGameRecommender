Board Game Geek

A huge thanks goes out to boardGameGeek.com for permitting me to use their data and make all of this possible. Board Game Geek is a giant website with about everything you can imagine relating to board games. They currently have about 6.5 million user game ratings and 80,000 games. I regularly use it and their community rated games to discover what game I want to play next, but one thing they don't actually have is a recommendation system. Using Board Game Geek's user game ratings and board game meta data, I have built a recommendation system!
